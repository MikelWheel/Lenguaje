Navegadores web

un navegador web (en inglés,web browser)es un software,aplicación o programa que permite el acceso a la Web,interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionabilidad básica de un navegador web es permite la visualización de docuemntos de texto, posiblemente con recurssos multimedia incrustados

Ejemplos de navegadores web

Safari:Safari es un navegador web de códgio cerrado desarrollado por Apple Inc.Está disponible para macOS,iPadOS e iOS.

Mozilla Firefox es un navegador web libre y de código abierto desarrollado para Linux,Android,iÓs,macOS y Microsoft Windowns coordinado por la Coporación Mozilla y la Fundación Mozilla.

Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
